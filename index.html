Meu nome é Vinicios Bucedi

Sou Cientista de Dados formado pela Universidade Estácio de Sá com Coeficiente de Rendimento de 9,39, com uma trajetória que mistura experiência prática em negócios e formação técnica sólida.

Antes de me dedicar à Ciência de Dados, trabalhei com gestão administrativa e análise financeira no varejo — o que me deu uma visão real de como dados impactam decisões de negócio, controle de custos e resultados. Meu objetivo é aplicar Python, SQL e Machine Learning para gerar insights que aumentem a lucratividade das empresas, transformando dados em decisões estratégicas.

Projetos em Ciência de Dados

📊 Análise de Retenção RFM – Varejo

📌 Sobre o projeto

Projeto de análise de dados focado em entender o comportamento de clientes utilizando segmentação RFM (Recência, Frequência e Valor Monetário) e análise de Cohort, aplicado a dados de varejo do dataset Superstore.

📈 O que foi feito

  • Segmentação de clientes com análise RFM.
  • Análise de Cohort para medir retenção ao longo do tempo.
  • Visualizações com Heatmap, Treemap, Gráficos de Linha e Barras.
  • Identificação de clientes em risco de churn.

🛠️ Ferramentas utilizadas

  • Power Query.
  • Fórmulas Avançadas (ÍNDICE, CORRESP, PORCENTRANK, MÁXIMO.SE).
  • Formatação Condicional (Heatmap).
  • Gráficos Dinâmicos (Treemap, Linha, Barras, Pizza).
  • Análise de Cohort e Segmentação RFM.
